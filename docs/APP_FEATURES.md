## Feature categories (for website sections)

### 1. Scheduling & dispatch

| Feature | Benefit (website copy) |
|---------|------------------------|
| **Task management** | Create, assign, and track cleaning jobs from mobile or web |
| **Calendar view** | See the week at a glance; tap any day to schedule |
| **Rota builder** | Drag-and-drop team assignment with leave awareness |
| **Recurring jobs** | Automate weekly/monthly property visits |
| **Route optimization** | Reduce travel time between properties |
| **Google Sheets sync** | Keep your master spreadsheet and app in sync both ways |
| **Real-time updates** | Status changes appear instantly across devices |

### 2. Field operations (cleaner app)

| Feature | Benefit |
|---------|---------|
| **Simple task list** | Cleaners see only today's jobs — no clutter |
| **Work timer** | Start, break, resume, submit — hours logged automatically |
| **GPS verification** | Location checked on start/complete; managers notified if off-site |
| **Live GPS tracking** | See cleaner location during active jobs |
| **Checklists** | Room-by-room tasks; must complete before starting (optional rules) |
| **Photo upload** | Before/after proof; works offline, syncs later |
| **Offline mode** | Full job completion without internet |
| **SOS safety button** | One-tap emergency alert to managers |
| **Job notes & journal** | Field notes attached to each property visit |

| **QA scoring** | Manager quality assurance workflow |

## Personas (for website “Who it’s for” section)

### Cleaning company owner
**Pain:** No visibility into operations, payroll disputes, client churn  
**TidyFlow gives:** Revenue dashboard, subscription control, reports, client proof

### Operations manager
**Pain:** Spreadsheets, WhatsApp chaos, no-shows, quality complaints  
**TidyFlow gives:** Rota, live tracking, AI assign, payroll approval, issue log

### Cleaner (field worker)
**Pain:** Complicated apps, no signal underground, unclear instructions  
**TidyFlow gives:** Simple job list, offline timer, photos, checklist, SOS

### Property / facility client
**Pain:** “Did they actually clean?”  
**TidyFlow gives:** Portal link, photos, PDF report, review form
